## Overview

Analyzed Brazilian e-commerce data as part of a 4-member capstone team at the University of North Texas, applying RFM analysis and K-Means clustering to calculate customer lifetime value and identify key customer segments. Delivered retention strategy recommendations supported by Python analysis and Tableau visualizations.

## Business Problem

E-commerce businesses struggle to identify which customers drive long-term value versus one-time purchases. Without segmentation, marketing spend is spread inefficiently across the customer base. This project builds a data-driven framework for targeted retention strategy.

## Methodology

1. **RFM Analysis** — scored customers on Recency, Frequency, and Monetary value
2. **K-Means Clustering** — grouped customers into distinct behavioral segments
3. **CLV Calculation** — estimated lifetime value per segment to prioritize retention spend
4. **Visualization** — Tableau dashboards communicating findings for a non-technical audience

## Key Outcome

- Identified distinct customer segments with differentiated CLV profiles
- Recommended targeted retention strategies per segment
- Visualizations designed for executive and non-technical stakeholder review
